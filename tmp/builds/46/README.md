# Show Your Work

This is a Next.js application designed to showcase a gallery of work with a contact form to reach out.

## Getting Started

1. Clone the repository.
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deployment

This application is ready to be deployed to platforms like Vercel, Netlify, or any server where you can run a Node.js application.
