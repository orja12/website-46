# Show Your Work

This is a Next.js application for showcasing your personal work through a gallery and providing a contact form.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build the Project

To build the project for production:

```bash
npm run build
```

You can start the built application using:

```bash
npm start
```

## Technologies Used

- Next.js
- Tailwind CSS
- TypeScript
- shadcn/ui
